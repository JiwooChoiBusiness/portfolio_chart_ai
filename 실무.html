🎯 문제 정의

1. 발견 배경

2023년 11월, MacBook M2 Pro (14/16 inch) 모델에서 "Freezing" 증상으로 입고된 기기 중, 수리 완료 후 7일 내 재입고되는 비율이 급증했습니다.

2. 문제의 심각성

  • 재수리율: 특정 에러 코드 케이스에서 약 70%
  • 비용 영향: 무상 재수리로 인한 부품 비용 + 부가 비용 + 인건비 손실
  • 고객 경험: 동일 문제로 재방문하는 고객 불만 증가
  • 운영 효율: 재수리 케이스가 신규 수리 처리 시간을 압박

3. 근본 원인 가설

초기 분석 결과, 테크니션들이 Freezing 증상에 대해 명확한 원인 파악 없이 LAS, LCD, Trackpad, Motherboard에 대해 무분별하게 교체.

4. 기존 SOP 기준 원인 파악

  • Freezing은 하드웨어/소프트웨어 모두에서 발생 가능한 증상
  • 기존 SOP에 M2 pro 모델에 대한 명확한 진단 프로세스 부재
  • 개인 할당량 달성을 위해 테크니션들이 "일단 교체하고 보자"는 방식으로 접근하기도 함
💡 핵심 문제 결론: "원인 미파악 상태의 무분별한 부품 교체로 인한 재수리율 급증"

# Python 가상코드 - Freezing 케이스 분석

import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

# 1. 데이터 불러오기
try:
    df = pd.read_csv('repair_data.csv')

    # 2. 데이터 전처리
    # 결측치 처리
    df['error_code'] = df['error_code'].fillna('')
    df['customer_comment'] = df['customer_comment'].fillna('')
    df['replaced_part'] = df['replaced_part'].fillna('Unknown')

    # 대문자 통일 및 공백 제거
    """
    중복방지는 수리 데이터 특성상 중복 데이터가 필요해서 하지 않았습니다"""
    df['model'] = df['model'].str.upper().str.strip()
    df['error_code'] = df['error_code'].str.upper().str.strip()

    # 3. 분석 기준 및 필터링 설정(실무에서 실제로 아래 keywords를 포함하여 찾았음)
    target_models = ['J414', 'J416']
    keywords = ['Freezing', 'Sleep mode', 'No LCD', 'No power']
    regex_pattern = '|'.join(keywords)

    # 모델(J414, J416) + 증상 키워드 필터링
    filtered_df = df[
        (df['model'].isin(target_models)) &
        (df['error_code'].str.contains(regex_pattern, case=False, na=False) |
         df['customer_comment'].str.contains(regex_pattern, case=False, na=False))
    ].copy()

    # 4. 부품별 재수리율 계산
    analysis_result = filtered_df.groupby('replaced_part')['re_repair'].agg(['mean', 'count']).reset_index()
    analysis_result.columns = ['Part', 'Rework_Rate', 'Total_Cases']
    analysis_result['Rework_Rate'] *= 100

    # 시각화를 위해 큰 순으로 정렬
    analysis_result = analysis_result.sort_values(by='Rework_Rate', ascending=False)

    # 5. 시각화
    plt.figure(figsize=(12, 7))
    sns.set_style("whitegrid")
    ax = sns.barplot(x='Part', y='Rework_Rate', data=analysis_result, palette='magma')

    plt.title('Analysis of Rework Rates for J414/J416 Freezing Issues', fontsize=16, pad=20)
    plt.ylabel('Rework Rate (%)')
    plt.ylim(0, 100)

    # 데이터 레이블 추가
    for i, p in enumerate(ax.patches):
        rate = analysis_result.iloc[i]['Rework_Rate']
        count = int(analysis_result.iloc[i]['Total_Cases'])
        ax.annotate(f'{rate:.1f}%\n(n={count})',
                    (p.get_x() + p.get_width() / 2., p.get_height()),
                    ha='center', va='bottom', fontsize=10, fontweight='bold')

    plt.tight_layout()
    plt.savefig('repair_analysis_report.png', dpi=300) # 고해상도 이미지 저장(ppt용)
    plt.show()

    print("Analysis_Result(Filtered)")
    print(analysis_result)

except FileNotFoundError:
    print("Cannot find the file. Please try again.")
except Exception as e:
    print(f"Error: {e}")
